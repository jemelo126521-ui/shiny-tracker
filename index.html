Configuration Supabase

Supabase est gratuit et permet aux widgets OBS de se mettre à jour en temps réel. Configuration unique.
1. Va sur supabase.com → compte gratuit → New Project
2. SQL Editor → New query → colle et exécute : CREATE TABLE IF NOT EXISTS state ( id TEXT PRIMARY KEY DEFAULT 'main', data JSONB NOT NULL DEFAULT '{}', updated_at TIMESTAMPTZ DEFAULT NOW() ); INSERT INTO state (id,data) VALUES ('main','{}') ON CONFLICT DO NOTHING; ALTER TABLE state ENABLE ROW LEVEL SECURITY; CREATE POLICY IF NOT EXISTS "all" ON state FOR ALL USING (true) WITH CHECK (true);
3. Settings → API → copie Project URL et anon public key
