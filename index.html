Configuration Supabase
Pour que les widgets OBS fonctionnent en temps réel, l'app utilise Supabase (base de données gratuite). Suis ces étapes une seule fois.
1. Va sur supabase.com → crée un compte gratuit → New Project
2. Dans ton projet → SQL Editor → colle et exécute ce SQL :

CREATE TABLE state ( id TEXT PRIMARY KEY DEFAULT 'main', data JSONB NOT NULL DEFAULT '{}'::jsonb, updated_at TIMESTAMPTZ DEFAULT NOW() ); INSERT INTO state (id, data) VALUES ('main', '{}'::jsonb); ALTER TABLE state ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON state FOR ALL USING (true) WITH CHECK (true);
3. Settings → API → copie Project URL et anon public key
